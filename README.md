# Amazon Shift Helper


A beginner Python project that estimates weekly pay based on hours worked and hourly pay rate.

## Why I Built This

I built this project because I work in warehouse operations and wanted a simple way to estimate weekly pay, including overtime.

## Features

- Calculates regular weekly pay
- Includes overtime after 40 hours
- Uses hourly pay rate
- Beginner-friendly Python code

## Skills Used

- Python
- User Input
- Conditional Logic
- Basic Math
